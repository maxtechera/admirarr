Features

Everything your fleet needs

One binary handles the full lifecycle — from first deploy to daily operation.

🚀

Zero to fleet in 15 minutes

12-phase guided setup deploys Jellyfin/Plex + all *Arr services, wires them together, and writes config — no manual clicking.

admirarr setup
🩺

34-check diagnostic engine

15 categories: connectivity, API keys, containers, VPN, media paths, indexers, hardlinks, disk, cross-service wiring, and more. AI-powered repair built in.

admirarr doctor --fix
📊

Fleet at a glance

Services, libraries, active downloads, disk usage, and pending requests — one command. Live-updating TUI for continuous monitoring.

admirarr status --live

Agent Skills compatible

Single binary, zero deps

JSON output everywhere

Every command supports -o json for structured output. Pipe to jq, feed to scripts, or let agents parse results programmatically.

Full media operations

Install

Get Admirarr

One binary, no runtime. Pick your platform.

Linux / macOS (curl)

curl -fsSL https://get.admirarr.dev | sh

Homebrew

brew install maxtechera/tap/admirarr

Go

go install github.com/maxtechera/admirarr@latest

Arch AUR

yay -S admirarr-bin

Debian / Ubuntu

dpkg -i admirarr_*.deb

Windows PowerShell

irm https://get.admirarr.dev/windows | iex
